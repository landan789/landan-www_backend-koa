
## 首次安装步骤

[1]. 步骤一，使用 git clone 下载此仓库代码
```shell
略
```
---------------------------------------

[2]. 步骤二，安装 Nodejs
```OS
略
```
---------------------------------------


[3]. 步骤三，在全域，安装 Yarn
```OS
略
```
---------------------------------------

[4]. 步骤四，在全域，安装 PM2
```shell
sudo yarn global add yarn
```
---------------------------------------


[5]. 步骤五，在项目根目录，安装 node_modules
```shell
yarn
```
---------------------------------------


[6]. 步骤六，在项目根目录，打包
```shell
yarn run build
```
---------------------------------------

[7]. 步骤七，在项目根目录，以 pm2 执行打包过的项目
```shell
pm2 start pm2.json
```
---------------------------------------

[8]. 步骤八，访问
```shell
curl 127.0.0.1:3000/
```
---------------------------------------
